1）声明
   使用 var 变量声明
	js代码执行分两步
	1.词法解析  先实行关键词
	2.语法解析  再检查代码是否有误
   使用 let 块级作用域
	1.不会提升变量声明
	2.声明的变量具有块级作用域所以不提升声明

   使用 const 声明常量 
	const π=3.1415926
	1.不可以改变π的值
	2.不具备声明提升


2）数据分为  （基本数据） 和 （引用数据）  类型 
	
	基本数据类型 number  string  boolean  undefined  NaN
		1.数据存放在 栈内
	引用数据类型 function  object null
		1.数据存放在 堆内
		2.有对应的包装对象


	数据类型的隐试转换 
	
	js是弱类型语言的特点，类型转换时是隐试类型转换

	两种写法

			属性名称：属性值，

5)构造函数 
	构造函数是用来创建 对象。
	
	构造函数也叫 实例化对象	

	构造函数与普通对象的区别在能不能 传参。
	
	构造函数可以随意命名，但行业形势为了区分是不是构造函数 统一默认规定构造函数的命名的首字母 大写
	
	重点：	构造函数有一个属性prototype  指向的是构造函数的原型对象
		实例化对象有一个属性 __porto__  指向的是实例化对象的构造函数的原型对象
		构造函数的原型对象有一个属性 constructor 指向的是原型对象的构造函数

	new 关键字的作用是创建对象的分为3步
		1.创建一个空的对象
		2.把构造函数内的属性都赋值给空的对象
		3.把空对象返回
