path--操作磁盘路径的内置模块
#### 用于获取文件名、后缀名、拼接路径...
```
var path = require('path');
```
1. 格式化路径 path.normalize(路径)
2. 获取代表文件夹的部分 path.dirname(路径)
3. 获取路径的最后一部分 path.basename(路径, [文件的扩展名])
######完整路径=path.dirname+path.basename
4. 获取文件的后缀名 path.extname(文件路径) ★
    以下三种情况返回空字符串
    - 路径中不带.
    - 只要一个.
    - 以.开头
5. 拆分路径(转对象) path.parse(路径)
6. 组合路径(转字符串) path.format(obj)
    - 当dir存在时，忽略root
    - 当base存在时，忽略ext和name
7. 判断是否是个绝对路径 path.isAbsolute(路径)
8. 拼接路径 path.join([p1,p2,...])
    - 使用当前系统的路径分隔符(/)将给定的路径段拼接到一起，再对结果进行规范化
9. 拼接路径(生成绝对路径) path.resolve([p1,p2,...]) ★★
    - 不传参：当前工作目录(命令执行的位置)所在的绝对路径
    __dirname:当前该变量所在的js文件的绝对路径
    - 传参：
        - 如果在拼接完给定的路径段后，还没有生成绝对路径，该方法会以当前工作目录的绝对路径再次进行拼接
        - 路径从右往左依次拼接，直到找到绝对路径就停止拼接，直接返回
